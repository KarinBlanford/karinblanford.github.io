Jukebox Project Part 1:


1. Create a new Java project called Jukebox_<LastName>_<FirstName>

2. In your new Jukebox project, create the following classes (capitalization is important):

a. Album.java

b. Song.java

c. JukeboxPlayer.java


3. In JukeboxPlayer.java:

a. Add a main method

b. Create a new Scanner object called ‘keyboard’

c. Print ‘Welcome to the Jukebox Player! Select a song.’ to the console


4. In Song.java:

a. Create the following instance variables. Use your Ninja.java file as a reference.

i. songTitle (String)

ii. songLength (double)

iii. songArtist (String)

iv. albumName (String)

b. Create the following functions in Song.java:

i. a constructor that accepts (title, length, artist) as arguments

ii. getters/setters for all instance variables

iii. a toString method that returns the string “<songTitle> is by <songArtist> and is <songLength> minutes long”


5. In Album.java:

a. Create the following instance variables:

i. songList (array of Song objects)

ii. albumArtist (String)

iii. albumGenre (String)

iv. albumName (String)

b. Create the following functions in Album.java:

i. a constructor that accepts (songList, albumArtist, albumGenre, albumName) as arguments

ii. getters/setters for all instance variables

iii. a toString method that returns the string “<albumGenre> music <albumName> is by  <albumArtist> and contains the following songs <songList>.” (Print each song on a new line with a number in front of it:

“Rock music album Room on Fire is by The Strokes and contains the following songs:

1. ‘What ever Happened?’

2. ‘Automatic Stop’

3. ’12:51’”
