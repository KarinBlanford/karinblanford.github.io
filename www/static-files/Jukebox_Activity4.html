1. Add the following function to your JukeboxPlayer.java file:
     public static void populateList() {
        String baseDir = System.getProperty("user.dir") + System.getProperty("file.separator") + "songfiles" + System.getProperty("file.separator");
        ArrayList results = new ArrayList();
        File[] files = new File(baseDir).listFiles();
        for (File file : files) {
            if (file.isFile()) {
                results.add(file.getName());
            }
        }
     }

  2. Replace your songfiles file with http://www.karin.io/www./static-files/songfiles.zip. Make sure to unzip the folder before adding it to your project. Feel free to rename song files (but don't change the extension).

  3. Add "ArrayList songLibrary = populateList();" to your main method. You can access an element in your songLibrary by calling songLibrary.get(index) where index is the position of the song you want to access. This method returns the String name of the song.

  4. Using Album and Song Objects, create a Jukebox for a user to play music. The Jukebox should be able to:
    1. List all the songs in the Jukebox
    2. Play a song given it's index in the song list
    3. (optional)List all the albums in the Jukebox
    4. (optional)List all the songs in an album
    5. (optional)Play a random song from an album
