1. Create a new Java project called NinjasAttack_YourName.


2. Create a new Java class called ‘GameConsole’ - include a main method. 


3. Create a new Java class called ‘Ninja’ in the same package as ‘GameConsole’. 


4. Add the following instance variables to ‘Ninja’ with the ‘private’ access modifier:


a. String ninjaName

b. int pointsRemaining


5. Create a constructor in the ‘Ninja’ class that accepts a String as a parameter and assigns it to the instance variable ninjaName created in step 4. Default the pointsRemaining instance variable to 100.


6. Create ‘getter’ methods for each of the instance variables we created in step 4 (getRemainingPoints, getNinjaName).


7. Create ‘setter’ methods for each of the instance variables we created in step 4 (setRemainingPoints, setNinjaName). 


8. Replace the assignments to the instance variables in the constructor in step 5 to use the setter methods in step 7.


9. Create a ‘toString’ method that returns a String output of a ninja in the following format:


Ninja Name: <ninja name here>

Points Remaining: <ninja’s remaining points here>


10. In the ‘GameConsole’ class create two instances of the Ninja class: one called ‘player1’ (Player 1) and the other ‘player2’ (Player 2). 


11. Print both of the Ninja’s to your console.
